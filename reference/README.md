# Reference Material

This tree contains non-supported historical material that remains useful for provenance, comparison, or documentation:

- baseline repositories such as `CNNCap`, `PCT-Cap`, and `GNN`
- upstream or vendor flow snapshots such as `OpenROAD-flow-scripts`
- archived scripts, plots, and example datasets that are not part of the supported `capbench` package

Nothing under `reference/` is packaged or treated as a supported public API.
