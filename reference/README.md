# Reference Material

This tree contains development scripts and historical material that remains useful for provenance, comparison, or documentation:

- baseline repositories such as `CNNCap`, `PCT-Cap`, and `GNN`
- upstream or vendor flow snapshots such as `OpenROAD-flow-scripts`
- development scripts, plots, and example datasets that are not part of the `capbench` package

Nothing under `reference/` is packaged or part of the public `capbench` API.
